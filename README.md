# Chat Management Web Application

A simple chat management web application built using Node.js, Express.js, MongoDB, and EJS.

## Features
- Create and manage chats
- Store chat data using MongoDB
- Server-side rendering with EJS
- Clean MVC structure

## Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- CSS

## How to Run Locally
1. Clone the repository
2. Run `npm install`
3. Create a `.env` file and add MongoDB URL
4. Run `node index.js`
5. Open browser at `http://localhost:3000`
